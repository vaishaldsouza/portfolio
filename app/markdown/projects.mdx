### Resume Screener Web App 📄
*Full-Stack Resume Screening & Ranking Platform*

[![GitHub](https://img.shields.io/badge/%2F-repo-grey?logo=github&labelColor=%23818181&color=%23008282)](https://github.com/vaishaldsouza)

A full-stack web application designed to automatically screen, analyze, and rank resumes against target job descriptions, providing an interactive HR panel.

- ⚙️ **Automatic Screening**: Automatically screens and ranks resumes using text analysis algorithms in Python.
- 📊 **HR Dashboard**: Features an HR dashboard for HR teams to search, filter, and shortlist candidate profiles based on skill match scores.
- 🔗 **Full-Stack Stack**: Built with React front-end, Node.js & Express.js API gateway, and MongoDB for database storage.
- 🔧 **Tech Stack**: React, Node.js, Express.js, Python, MongoDB, REST APIs

### Movie Recommendation System 🎬
*Desktop GUI Recommendation Platform*

[![GitHub](https://img.shields.io/badge/%2F-repo-grey?logo=github&labelColor=%23818181&color=%23008282)](https://github.com/vaishaldsouza)

A GUI-based movie recommendation system that recommends movies based on user-selected genres, storing all backend movie metadata in SQL.

- 🎨 **Desktop GUI**: Interactive GUI layout developed using Python's Tkinter library.
- 🗄️ **Relational Database**: Leverages MySQL for backend data storage, queries, and genre relationship mapping.
- 🔧 **Tech Stack**: Python, Tkinter, MySQL
