### DataForge Studio 🔬
*Full-Stack ML Data Preparation Platform*

[![GitHub](https://img.shields.io/badge/%2F-repo-grey?logo=github&labelColor=%23818181&color=%23008282)](https://github.com/Prajwal18py/SMART-CSV-HEALTH-CHECKER) [![Live Demo](https://img.shields.io/badge/Live-Demo-green?logo=streamlit&labelColor=%23ff4b4b&color=%23008282)](https://dataforge-studio.streamlit.app/)

A production-grade web application enabling **end-to-end ML workflows** — from raw data ingestion to deployable model export.

- 🧩 **13 specialized data processing modules** for complete ML pipelines
- 🤖 AI-powered anomaly detection using **Isolation Forest** + advanced missing data imputation (MICE algorithm)
- ⚖️ Class balancing via **pure NumPy SMOTE/ADASYN** (no external imbalance libraries)
- 🏎️ Automated model training supporting **12 ML algorithms** with real-time data lineage tracking
- 📦 Handles **100,000+ row datasets** with streaming ops; supports CSV, Parquet, JSON, Excel
- 📄 Dark-themed PDF report generation using ReportLab
- 🔧 **Tech Stack**: Python, Streamlit, Pandas, Scikit-learn, Plotly, SQLite, Supabase

### CODEVIS 🧠
*AI-Powered DSA & OOP Learning Platform*

[![GitHub](https://img.shields.io/badge/%2F-repo-grey?logo=github&labelColor=%23818181&color=%23008282)](https://github.com/Prajwal18py/codevis) [![Live Demo](https://img.shields.io/badge/Live-Demo-green?logo=vercel&labelColor=%23000000&color=%23008282)](https://codevis.vercel.app)

A full-stack AI learning platform for CS students built with React 18, Groq LLaMA 3.3 70B, and Supabase.

- ⚔️ **Group Battle Mode** — up to 10 players compete in real-time coding battles, AI ranks all solutions
- 🤖 **LeetCode AI Tutor** — step-by-step code analysis with logic walkthrough, dry run, and complexity
- 🔍 **OOP Analyzer** — detects all 12 OOP concepts in C++ / Python with interactive class diagram
- 📊 **Algorithm Visualizer** — 14+ algorithms animated with C++ / Python code toggle
- 🌲 **Graph & Tree Visualizer** — BST traversals + interactive graph BFS/DFS
- 🧪 **AI Quiz** — generates 5 MCQs on any CS topic, scores saved to dashboard
- 📄 **AI Cheat Sheet** — generates printable cheat sheets with C++ or Python code
- 🔗 **Shareable Links** — share any analysis publicly, no login required
- 🔧 **Tech Stack**: React 18 + Vite, Groq LLaMA 3.3 70B, Supabase, Piston API, Vercel
- 📈 **Stats**: 11 AI modes · 100 battle problems · 10 max players · 4 themes

### Neural Brief 🧠
*Automated AI News Intelligence Platform*

[![GitHub](https://img.shields.io/badge/%2F-repo-grey?logo=github&labelColor=%23818181&color=%23008282)](https://github.com/Prajwal18py/Neural-Brief) [![Live Demo](https://img.shields.io/badge/Live-Demo-green?logo=vercel&labelColor=%23000000&color=%23008282)](https://neural-brief-eight.vercel.app/)

A fully automated AI news platform tracking 11 top sources, curating weekly digests with Groq LLaMA 3.3 70B, and delivering personalized briefings to subscribers every Friday.

- 📡 **Live Daily Feed** — Today's Top 3 AI stories updated every 3 hours from 11 sources including Google DeepMind, Anthropic and MIT Tech Review
- 🔥 **Signal Score** — every story rated 1-10 by AI based on importance, novelty and industry impact
- 🎭 **AI vs Reality** — cuts through media hype with an honest one-line reality check per story
- 🧩 **Break it down** — simplifies any story for beginners with a plain English explanation
- 🌍 **Reader Personalization** — "Why it matters" dynamically tailored for Students, Developers, Founders and Creators
- 📬 **Automated Friday Digest** — 15 curated stories with TL;DR, summaries and LinkedIn post generator delivered via email
- 🗄️ **Archive Page** — browse all past weekly briefs with full story breakdowns
- 🔧 **Tech Stack**: React 18 + Vite, Groq LLaMA 3.3 70B, Supabase, Brevo SMTP, Vercel Cron
- 📈 **Stats**: 11 sources tracked · 15 stories weekly · 4 reader personas · ₹0/month running cost
